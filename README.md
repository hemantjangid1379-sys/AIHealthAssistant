# AI Health Assistant

## 📌 Description
AI Health Assistant is a web-based application built using Java and Maven that analyzes user symptoms and predicts possible diseases.

## 🚀 Features
- Symptom-based disease prediction
- Web interface using JSP/Servlets
- Basic machine learning logic

## 🛠️ Technologies Used
- Java
- JSP / Servlets
- Maven
- HTML / CSS / JavaScript

## 📂 Project Structure
- src/main/webapp → Frontend (JSP, HTML, CSS, JS)
- Java classes → Backend logic
- pom.xml → Maven configuration

## ⚠️ Note
Sensitive data like API keys are not included in this repository.
